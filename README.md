# GradientLab - CSS Gradient Generator

A professional CSS gradient generator tool built with Next.js 16, TypeScript, and Tailwind CSS v4.

## Features

- **Multiple Gradient Types** — Linear, Radial, and Conic gradients
- **Color Stops** — Add, remove, and adjust color stops with position control
- **Angle Control** — Fine-tune gradient angles with an intuitive slider
- **Live Preview** — Real-time gradient preview as you adjust settings
- **CSS Output** — Copy-ready CSS code with one-click copy
- **Random Gradients** — Generate beautiful random gradients instantly
- **Preset Gallery** — 12 curated gradient presets to get started quickly
- **Responsive** — Works seamlessly on desktop and mobile

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS v4
- Lucide React (icons)
